Añadiendo JavaScript: Gestión de Imágenes (Lightbox)

2 de noviembre de 2025

Para cumplir con los requisitos de la actividad, el profesor solicitó en clase añadir interactividad con **JavaScript**, específicamente para la gestión de imágenes.

Hemos implementado una funcionalidad clásica y elegante conocida como **"Lightbox"**. En lugar de simplemente mostrar una imagen, ahora podemos hacer clic en ella para que se amplíe y ocupe el centro de la pantalla, mientras un fondo oscuro desvanece el resto del contenido.

¿Cómo se ha implementado?

La implementación ha requerido tocar nuestros tres tipos de archivos:

  1. `lightbox.js` (Nuevo): Se creó un nuevo archivo JavaScript. Este script espera a que la página cargue, busca todas las imágenes (``) dentro de los artículos (`.blog-post`) y les añade un "escuchador" de eventos (un addEventListener). Al hacer clic, el script activa el modal.
  2. `style.css` (Modificado): Añadimos nuevos estilos al final del CSS para definir el aspecto del modal (#lightbox). Por defecto, está oculto (display: none;), y el JavaScript lo hace visible (display: flex;) al añadir la clase .visible.
  3. Modificado entrada-3.html: Ahora enlaza al script justo antes de cerrar el </body>, usando la etiqueta <script src="lightbox.js" defer></script>.
